# 🏥 Hospital Management System

A complete web-based Hospital Management System with QR code payments, doctor appointment scheduling, lab test booking, patient health tracking, and SMS notifications.

## ✨ Features

### For Patients
- 📝 User Registration & Login
- 👤 Profile Management with Medical History
- 📅 Book Appointments with Doctors
- 🩺 AI-Based Symptoms Checker
- 💳 QR Code UPI Payments
- 📋 Digital Prescriptions (PDF)
- 📊 Health Dashboard (BMI, BP, Heart Rate)
- 🔬 Lab Test Booking (9 AM - 9 PM)
- 📎 Upload Medical Reports
- ⭐ Doctor Feedback & Ratings
- 📱 SMS Notifications

### For Doctors
- 👨‍⚕️ Dashboard with Today's Schedule
- 📋 Patient Queue Management
- 💊 Digital Prescription Generator
- 💰 Consultation Fee Management
- 📊 View Patient Feedback & Ratings
- 📑 Review & Verify Reports
- 📅 Schedule & Leave Management

### For Admin
- 📊 Complete Analytics Dashboard
- 👥 User Management
- 📅 Appointment Management
- 👨‍⚕️ Doctor Schedule Management
- 🔬 Lab Test Management
- 📈 Advanced Charts & Graphs
- 💰 Revenue Tracking
- 📤 Export Data (Excel/CSV)
- 🧪 Lab Booking Management

## 🛠 Technology Stack

| Technology   | Version | Purpose                |
|--------------|---------|------------------------|
| Python       | 3.13+   | Backend Language       |
| Flask        | 2.3+    | Web Framework          |
| SQLAlchemy   | 2.0+    | ORM Database           |
| SQLite       | 3       | Database               |
| Bootstrap    | 5.3     | Frontend CSS           |
| jQuery       | 3.6+    | JavaScript Library     |
| Chart.js     | 4.4+    | Analytics Charts       |
| QRCode       | 7.4+    | QR Generation          |
| ReportLab    | 4.0+    | PDF Generation         |
| Twilio       | 8.10+   | SMS Notifications      |
| Pandas       | 2.1+    | Data Export            |
| OpenPyXL     | 3.1+    | Excel Export           |
